Principais tags de HTML

Tags DescriçãoExemplos
<!DOCTYPE> Primeira tag a ser digitada em seu código. Serve para informar ao navegador a linguagem de seu código. Como estamos tratando de HTML, o exemplo utilizado será dessa linguagem. <!DOCTYPE html>
<html>

</html>		 Indica o início e o fim do seu código. Todo o código deve ser escrito entre as respectivas tags de abertura e fechamento. Como a tag <html> indica o início, deve-se complementá-la informando o idioma de sua página utilizando o termo "lang". Isso facilita a classificação de seu site em bancos de dados (do Google, por exemplo). <html lang="pt-br">

</html>
<head>

</head>		 É o "cérebro" de sua página, onde o título e as diretrizes devem ser informadas, tais como os arquivos css, jquery, javascript, entre outros. <head>

</head>
<meta/> Metadados. Utilizado para esclarecer ao navegador coisas que o HTML não consegue, como, por exemplo, o método de formatação do texto. O método aqui utilizado é o "UTF-8" e, para isso, utiliza-se o parâmetro "charset". Como <meta> é um parâmetro que não possui tag de fechamento utiliza-se a "/" antes de seu próprio fechamento. <meta charset="UTF-8"/>
<title>

</title>		 O título de sua página. Informação que será descrita na aba de seu navegador. Deve ser escrita dentro da tag head. <title> Hello, World! </title>
<body>

</body>		 A tag body; delimita o corpo do texto. Deve ser escrita dentro da tag <head> e, com exceção de algumas, todas as demais tags serão escritas dentro desta. Com a tag body, o esqueleto do código HTML está pronto para ser exibido em um navegador. <body>

</body>

Exemplo nº 01

Se você entendeu até aqui, seu código deve estar parecido com isso:

 <!DOCTYPE html>

 <html lang="pt-br">

   <head>

     <meta charset="UTF-8"/>
     <title> Hello, World! </title>

   </head>

   <body>

       Concluída a primeira etapa!!!

   </body>

 </html>

Se o seu código está idêntico ao exemplo acima, basta salvar o resultado utilizando a extensão ".html" e abrir o arquivo em seu navegador. Se você está utilizando o editor Notepad++, que foi o utilizado para a construção deste site e também o navegador Google Chrome, os quais eu recomendo, basta pressionar "shift + ctrl + alt + R" que o resultado da programação será mostrado no navegador Chrome. Lembre-se que para toda e qualquer alteração feita no código é necessário salvá-lo antes de rodá-lo. Caso contrário, não surtirá efeito.

Note que a frase no "body" de sua página aparece na formatação padrão do navegador e o títlo da página, "Hello, World!", aparece corretamente. Logo abaixo você aprenderá a formatar o texto de sua página. Por enquanto, sugiro você agora a praticar um pouco. Experimente, por exemplo apagar a linha onde você informou ao navegador qual a formatação utilizada em seu código. Para quem não se lembra, esse parâmetro foi "charset". Apague essa linha, salve o documento e atualize a página do navegador para ver o resultado.

Note, portanto, que para programar um código exige-se, além do conhecimento de linguagem, a chamada de lógica de programação, pois as vezes um simples ";" ou um "<", ou a falta deles, pode comprometer todo seu trabalho.

Pronto para dar início à personalização do seu código? Vamos lá.

Com essas tags, se você prestou bastante atenção, já é possível deixar os textos de sua página com uma aparência muito melhor. Tente escrever por conta uma página qualquer, salve o editor de texto e atualize a página de seu navegador para verificar o resultado.

A tag <div> merece um destaque muito grande e você entenderá isso quando estudar CSS. Quando for criar seu site, imagine que ele tenha várias divisões como se ele fosse criado em blocos que se encaixam perfeitamente. Não há uma regra para utilização de div's, mas sua correta utilização poupar-lhe-á bastante tempo.

IMPORTANTE! Lembre-se de fazer pequenas alterações, salvar e verificar o resultado. Nunca faça muitas alterações de uma só vez antes de verificar o resultado, pois se você errar talvez fique bem complicado de localizar o erro.

Dando valores aos atributos id e class

Não há obrigatoriedade em utilizar esses atributos quando se programa em HTML. No entanto, conforme dito anteriormente, se bem utilizados eles podem fazer muita diferença ao seu trabalho. A principal delas é economia de tempo. Você verá que após feita a divisão, identificação e classificação dos parâmetros, seu trabalho será muito bem compensado.


id

O atributo id serve para identificar as partes de seu código, permitindo manipulações do tipo "ir até o topo" ou "ir a determinado local da página", bem como tratamento por meio de outras linguagens, como JavaScript, por exemplo.

Um exemplo de modelo JavaScript poderia ser uma simples questão de matemática:

Exemplo nº 05

<!DOCTYPE html>
<html>

 <head>
  <meta charset="UTF-8"/>
 </head>

 <body>
  <h1 id="exemplo">Quanto é 5 + 5?</h1>

    <button onclick="displayResult()">Resposta</button>

  <script>
   function displayResult() {
     document.getElementById("exemplo").innerHTML = "10";
    }
  </script>

 </body>
</html>

Note que para a função JavaScript "document.getElementById" funcionar foi necessário estabelecer uma ligação com o código HTML, que, neste caso, foi feita por meio do atributo id. A leitura do código no exemplo acima seria algo do tipo: ao clicar no botão de função "displayResult", localizar o elemento do texto que tem a Id "exemplo" e substituir o conteúdo HTML desse Id por "10".


class

O atributo class permite customizar sua página. Com ele é possível definir o estilo de fonte, cor do fundo, parágrafo etc. de blocos inteiros, separados por div's e demais tags.

Ou seja, é possível configurar uma vez só o estilo do texto que você quer que apareça em seu site ao invés de ter que defini-lo parágrafo por parágrafo.

Para esclarecer melhor essa questão, demonstraremos no exemplo abaixo como customizar seu texto de duas formas. Uma menos trabalhosa e outra inviável. Veja:

Exemplo nº 04

 <!DOCTYPE html>
 <html lang="pt-br">
   <head>
     <meta charset="UTF-8"/>
     <title> Customizando seu texto do jeito certo </title>
     <style>
       div.colors {
        background-color: rgba(215, 0, 0, 0.3);
        color: black;
        margin-left: 30%;
        margin-right: 30%;
        text-align: justify;
       }
       div.colors2 {
        background-color: rgba(0, 0, 0, 0.9);
        color: white;
        margin-left: 30%;
        margin-right: 30%;
        text-align: justify;
       }
     </style>
   </head>
   <body>
       <div class="colors">
        <p>Fazendo dessa forma você precisa configurar uma só vez a forma como quer que seu código apareça.</p>
        <p>Da próxima vez que quiser que o texto tenha a mesma formatação basta atribuir o nome da classe para a div correspondente.</p>
       </div>
       <div class="colors2">
        <p>Vamos fazer a mesma coisa sem utilizar as classes, mas utilizando a tag span.</p>
       </div>

   <br/>

       <div style="margin-left: 30%; margin-right: 30%; text-align: justify; background-color: rgba(215, 0, 0, 0.3);">
        <p><span style="color: blue;">Imagine o trabalho ter que escrever ou copiar e colar esse código toda vez que desejar formatar um texto dessa forma? Imagine quando quiser dar uma mudada em seu site. Qual seria mais prático?</span></p>
       </div>

       <div style="margin-left: 30%; margin-right: 30%; text-align: justify;">
<p><span style="background-color: black; color: white;"> Note, todavia, que é possível formatar o texto de forma diferente, mas você pode classificar seu código em várias classes e utilizar a formatação pré-definida para cada uma delas sem ter que copiar e colar o código de formatação toda vez que precisar.</span></p>
       </div>
   </body>
 </html>

Perceba que o estilo do texto é feito no campo "head", que é o cérebro de seu código. Da forma como feito significa que tudo que está dentro de uma div de classe "colors" receberá a formatação dada a "div.colors". O mesmo ocorre com o que estiver dentro de "div.colors2".

Uma observação importante é que mesmo classificada uma div, se por acaso forem atribuídos estilos específicos à tag, o resultado se mostrará de acordo com os estilos específicos e não como os estilos da respectiva div.

Por exemplo, mesmo que a "div.colors" estabeleça a cor da fonte como preta, se por acaso for atribuído um "span style="color: blue" dentro da tag <p>, o texto sairá azul e não preto.
